# Title: Etch A Sketch

An interactive drawing grid inspired by an Etch-A-Sketch. Hover over squares to have them be filled or change color. Change the size of the grid if you want.

## Project Notes

- Container keeps the same total area
- Can only input up to 100

## Key Features

- Event handlers for user interactions
- DOM and UI maninpulation via JS
- Flexbox manages the grid
